About JoggAI

Built for the messy middle between an idea and a finished video.

Based in Singapore, JoggAI focuses on making video creation simpler, faster, and easier to repeat. The platform removes repeated production steps so marketers, creators, startups, agencies, and teams can spend more time on the story, audience, and message.

Its practical premise is straightforward: start from the product link, campaign idea, images, podcast clip, or rough script already available, then make the path to a usable video shorter.

01

Core workflow bundle

Jogg Skills

Install the repository Skills for Jogg API, lip sync, and trend-to-video workflows.

npx skills add JoggAI-Tech/jogg-skills

02

Optional plugin · same repository

Smart Slides

Add the bundled Smart Slides plugin when you want local planning, editing, preview, and rendering with Jogg assets.

  1. Register the JoggAI marketplace
    codex plugin marketplace add JoggAI-Tech/jogg-skills --ref main

  2. Install Smart Slides from that marketplace
    codex plugin add smart-slides@jogg-skills
